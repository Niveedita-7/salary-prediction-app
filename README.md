# salary-prediction-app
# 💼 Salary Prediction App

## 📌 Project Description

This project is a Machine Learning-based web application that predicts employee salaries using key features such as company rating and number of salaries reported.

It demonstrates the complete workflow of a data science project including data preprocessing, model training, and deployment using Streamlit.

---

## 📊 Dataset

The dataset contains the following columns:

* Rating
* Company
* Job Title
* Salary
* Salaries Reported
* Location
* Employment Type
* Job Roles

### Features Used:

* Rating
* Salaries Reported

### Target Variable:

* Salary

---

## ⚙️ Model Used

* Linear Regression
* StandardScaler for feature scaling

---

## 📁 Files in this Repository

* `Salary_Dataset.xls` → Dataset used for training
* `training.ipynb` → Model training notebook
* `model.pkl` → Trained machine learning model
* `scaler.pkl` → Scaler used for preprocessing
* `app.py` → Streamlit web application
* `requirements.txt` → Required libraries

---

## ▶️ How to Run the Project

### 1. Install dependencies

```
pip install -r requirements.txt
```

### 2. Run the Streamlit app

```
streamlit run app.py
```
